# Blog

## About

This repo is a blog which uses the `basic-ssg` framework. Its a static website builder, which provides plugins and utilities to build a site.

It has the following structure

- Main aside navigation, present in all pages
This aside navigation should have two main menus: areas e ferramentas. Areas are the main medical areas: clinica medica, cirurgia, pediatria, ginecologia, etc. Ferramentas are the tools that the blog provides: calculadora, etc.

Each area should have its own page, with a list of articles related to that area. Each article should have its own page, with the article content.

- Footer present in all pages

Footer which is a simple copyright claim

- Primary color is tailwind blue 600 (oklch(54.6% .245 262.881)), and uses shades of black and white

- Articles have tags, and use MD with brain-matter for basic strucutre. For a more advanced structure, its possible to pass iframes as components

- IT should be fully composable, so header, footer, head, etc. All should be components.
